## $name$

### Prerequisites

1. Java 8
2. Scala 2.11.3
3. SBT 0.13.0+

### Getting Started

Inside the root directory of the repository:

1. Launch SBT:

        $ sbt

2. Compile everything and run all tests:

        > test

3. Start the application:

        > re-start

4. Browse to [http://localhost:8080](http://localhost:8080/)

5. Stop the application:

        > re-stop
